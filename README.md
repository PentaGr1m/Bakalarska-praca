## Bakalárska práca – Návrh informačného systému pre knižnice

Systém obsahuje:
- prihlasovanie a odhlasovanie zamestnancov do systému  
- spravovanie výpožičiek (vytváranie, predlžovanie a pod.)  
- správu kníh vlastnených knižnicou (pridávanie, vyraďovanie a pod.)  
- správu čitateľských preukazov
